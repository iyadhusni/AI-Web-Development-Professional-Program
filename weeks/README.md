# Course Sessions — `weeks/`

Weekly lesson plans, starter code, and project resources for the AI Web Development Professional Program.

## Structure

Each week folder contains:
- **Session directories** — `01-*` and `02-*` with lesson READMEs, code examples, and AI prompts
- **Project folders** — Starter code and scaffolding for in-class exercises
- `README.md` — Week overview with topics, project table, and links to sessions

## Starter Projects

Starter code for in-class and homework projects lives alongside session directories. Each starter provides a minimal scaffold so students can focus on the lesson's core concept. Most include commented CDN links, HTML structure, and CSS/JS placeholders ready for modification.

## AI Integration

Every session includes AI prompt templates for code generation, debugging, refactoring, and content creation. See [`../tools-reference.md`](../tools-reference.md) for the full tools catalogue.
