## Evaluation Mechanism

The evaluation mechanism for the AI Web Development Professional Program at Acadima College is based on practical web development tasks, short knowledge checks, AI-assisted coding activities, and a final portfolio-based web project.

The aim is to assess both technical understanding and practical ability to build, improve, and present real frontend web projects using HTML, CSS, JavaScript, modern development tools, and AI-assisted workflows.

### 1. Phased Evaluation — 30%

30% of the final grade is distributed across practical tasks, mini projects, short tests, and participation in AI-supported development activities throughout the program.

#### Phased Projects

Phased projects are designed to help learners build confidence step by step before reaching the final project stage. They may include:

- Building a personal bio page using HTML.
- Styling web pages using CSS, colour palettes, typography, and layouts.
- Creating responsive pages using Flexbox and Grid.
- Building interactive components with JavaScript.
- Developing a mini project such as a To-Do app, quiz, theme switcher, or weather app.
- Using version control and deployment platforms.
- Applying accessibility and responsive design principles.
- Using AI tools to support planning, coding, debugging, refactoring, and improvement.

#### Short Tests

Short tests are used to measure the learner's understanding of key concepts covered during the program. They may assess:

- HTML structure and semantic elements.
- CSS styling, layout, and responsive design.
- JavaScript fundamentals and DOM interaction.
- Version control and deployment basics.
- Accessibility standards.
- Safe and effective use of AI tools in web development.
- Preparation for industry-recognised frontend development certification.

Submitting all phased projects and completing short tests is an essential requirement for completing the program requirements.

### 2. Final Portfolio Project — 70%

The final portfolio project is the main foundation for evaluating learner performance in the AI Web Development Professional Program. Learners will prepare and present a complete responsive portfolio website that reflects the skills and knowledge gained throughout the program.

The final project should demonstrate:

- Clear website structure using HTML.
- Professional styling using CSS and/or a CSS framework.
- Responsive design for desktop, tablet, and mobile screens.
- JavaScript interactivity.
- Accessibility best practices.
- Clean and organised code.
- Effective use of AI tools for planning, debugging, content improvement, and code refinement.
- A professional portfolio presentation suitable for career and interview use.

The final project is evaluated based on clear criteria including quality, functionality, user experience, accessibility, creativity, technical accuracy, and the extent to which the project objectives are achieved.

The final portfolio project represents **70% of the final grade**.

The final project must be discussed in front of the evaluation committee through a professional presentation and explanation of the project details.

### Important Notes

- Failure to submit the final portfolio project or failure to discuss it in front of the evaluation committee will result in the learner receiving only an attendance certificate without a grade.
- Adhering to phased project submission deadlines and short test dates is required to ensure fair and comprehensive evaluation.
- Learners will be provided with timelines, project briefs, submission requirements, and detailed guidance for each project and test to support the best possible performance.
- The use of AI tools is encouraged as part of the learning process; however, learners must be able to explain their code, justify their design choices, and demonstrate personal understanding of the work submitted.

### Student Guidelines

- Learners should invest effort in phased projects and short tests to build a strong foundation for the final portfolio project.
- Learners should follow the guidance and feedback provided by trainers to improve code quality, design consistency, accessibility, responsiveness, and user experience.
- Learners should use AI tools responsibly to support learning, not to replace understanding.
- Learners should prepare and discuss the final portfolio project professionally, showing how they applied frontend development skills and AI-assisted workflows throughout the program.
- Learners are encouraged to keep their code repository profile, project links, and portfolio updated during the program to support career readiness.
