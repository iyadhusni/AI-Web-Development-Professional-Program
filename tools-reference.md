## Recommendation Guide

### By Budget
- **Free stack:** OpenCode + Google Stitch + Gemini (free) + GitHub Pages
- **Budget premium ($10-20/mo):** VS Code + GitHub Copilot + ChatGPT Plus + Netlify
- **Full premium ($40-60/mo):** Cursor + Claude Pro + v0 + Vercel

### By Role
- **Developer-focused:** OpenCode / Claude Code + VS Code + Copilot + GitHub Pages
- **Design-focused:** Google Stitch + Figma + Framer + Netlify
- **Full-stack curious:** Lovable / Bolt.new + Firebase Studio + Vercel
- **Career-focused:** ChatGPT Plus + GitHub Copilot + GitHub Pages portfolio

### By AI Provider Ecosystem
- **Google ecosystem:** Gemini (chat) + Google Stitch (design) + Gemini CLI / Antigravity CLI (terminal) + Firebase Studio (deploy)
- **Anthropic ecosystem:** Claude (chat) + Claude Code (terminal agent)
- **OpenAI ecosystem:** ChatGPT (chat) + Codex CLI (terminal) + GitHub Copilot (IDE)
- **Alibaba ecosystem:** Qwen Chat + Qwen Code + Alibaba Cloud deploy
