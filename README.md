# AI Web Development Professional Program

**Acadima College** — 11-week intensive frontend web development program with AI-assisted workflows.

## Program Overview

- **Duration:** 11 weeks (27 June – 9 September 2026)
- **Schedule:** 2 sessions/week — Saturday (3 hrs) + Wednesday (2 hrs)
- **Focus:** HTML, CSS, JavaScript, Git/GitHub, CSS Frameworks, Portfolio Building, Accessibility
- **AI Integration:** AI-assisted coding, debugging, refactoring, and career prep throughout

## Curriculum

| Module | Topics |
|--------|--------|
| Weeks 1–2 | Orientation, AI Tools, HTML, CSS, Flexbox & Grid |
| Weeks 3–4 | JavaScript Fundamentals, DOM Manipulation, Mini Projects |
| Weeks 5–6 | Git/GitHub, Deployment, CSS Frameworks |
| Weeks 7–8 | Clone Challenge |
| Week 9 | Portfolio Planning & Polish |
| Week 10 | Accessibility, Final Review |
| Week 11 | Exam Prep, Career Prep |

## Instructor Hours

| Instructor | Hours | Topics |
|------------|-------|--------|
| **Dr Iyad** | **31 hrs** | AI Tools, CSS, Flexbox & Grid, JavaScript, Advanced Styling, Portfolio |
| **Dr Motaz** | **24 hrs** | HTML, Git & Deployment, Clone Challenges, Accessibility, Exam Prep, Career Prep |
| **Total** | **55 hrs** | 11 weeks × 5 hrs/week |

## Evaluation

- **Phased Evaluation (30%):** Practical tasks, mini projects, short tests, AI-supported activities
- **Final Portfolio Project (70%):** Complete responsive portfolio website presented to evaluation committee
